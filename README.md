# Vue modal manager

Modal manager for Vue 3 applications

## Installation

### With npm

```sh
npm install vue-modal-manager
```

### With yarn

```sh
yarn add vue-modal-manager
```

### With pnpm

```sh
pnpm add vue-modal-manager
```
